WebView Renewal

Language : Objective-c

Framework : WebKit, UIKit

🍦 WebView 통합 : 게임내 공지사항, 고객센터, 가이드 등과 같은 총 9개의 WebView 관련 framework들의 공통 코드를 하나로 묶어 상속구조로 변경. 공통 쿠키값, WKWebView Configuration정의. 공통 요구사항 구현이나 신규 OS대응 시 하나의 framework 수정 만으로 모든 웹뷰가 반영되어 리소스 절감.

🍦 디자인, 기능 Renewal : 기기와 orientation에 맞게 디자인된 View를 노출할 수 있도록 변경. 웹뷰 내에서 결제, 웹뷰에서 다른 웹뷰를 노출하기 등의 기능 구현.

Auto-Renewing subscriptions

Language : Objective-c

Framework : StoreKit

자동 갱신 구독상품(Auto-renewable subscriptions) 구현

🍦 일반 소모성 상품(Consumable)만을 지원하던 로직에 구독상품을 위한 로직을 추가하고, 기기 변경이나 유저 식별자 변경 등을 고려한 설계 및 개발 진행

🍦 StoreKit에 대한 기본지식이 없는 상태에서 시작한 프로젝트로, Apple의 공식 Programming Guide를 정독하면서 테스트

🍦 결제 서버와의 많은 커뮤니케이션으로 다양한 예외상황을 파악하고 대응할 수 있도록 처리

Email Authentication

Language : Objective-c

Framework : WebKit, UIKit

Facebook, Twitter와 같은 3rd party 인증수단이 아닌 회사 자체 인증수단을 제공하기 위한 Email Authentication Framework 개발

🍦 1차 개발: View가 없는 api로, 가입과 로그인 기능 설계 및 개발. email 과 password를 parameter로 받아 암호화하여 전송하고, 로그인 이후 만료시간을 확인하여 자동로그인을 진행하는 기능

🍦 2차 개발: 게임 개발사 요구사항을 받아들여 View를 함께 제공, 웹뷰 내에서 사용자가 회원가입/로그인/계정관리 등이 가능하도록 개발

🍦 계정관리 페이지, 복구 이메일 미등록 유저 페이지 추가 개발
